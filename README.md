## Deskripsi
Proyek ini merupakan pengembangan Learning Management System (LMS) berbasis Moodle dengan integrasi fitur Kanban Board dan Reflection Checkpoint untuk meningkatkan kemandirian belajar peserta didik. Fitur Kanban Board digunakan sebagai media manajemen tugas visual yang membantu siswa memantau progres pekerjaan secara terstruktur melalui tahapan To-Do, In Progress, dan Done, sedangkan Reflection Checkpoint berfungsi sebagai sarana refleksi belajar dan evaluasi pemahaman sebelum siswa melanjutkan ke tahap pembelajaran berikutnya. Sistem juga menerapkan mekanisme lock checkpoint sehingga akses tugas tertentu hanya dapat dibuka setelah aktivitas refleksi diselesaikan. Melalui integrasi fitur ini, proyek bertujuan menciptakan lingkungan pembelajaran digital yang lebih interaktif, terorganisir, dan mendukung pengembangan self-regulated learning serta kemampuan manajemen proyek peserta didik.
